# Проектирование системы с использованием UML

## Описание проекта

Целью проекта было спроектировать программную систему с использованием методологии UML.
В рамках работы были разработаны различные типы диаграмм, описывающие структуру и поведение системы.

Тема проекта:
Система управления рабочими сменами сотрудников розничных магазинов

## Использованные диаграммы

В проекте были созданы следующие UML-диаграммы:

- Диаграмма вариантов использования (Use Case Diagram)
- Диаграмма активности (Activity Diagram)
- Диаграмма классов (Class Diagram)
- Диаграмма состояний (State Diagram)
- Диаграмма последовательностей (Sequence Diagram)

Все диаграммы находятся в папке `/diagram`.

## Использованные инструменты

- UML
- Visual Paradigm
- Git

## Что демонстрирует данный проект

- Навыки системного анализа
- Объектно-ориентированное проектирование
- Работа с UML-диаграммами
- Понимание архитектуры программных систем
